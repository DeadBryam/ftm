# Icons

Place your icon files here:

- `32x32.png` - Small icon
- `128x128.png` - App icon
- `128x128@2x.png` - Retina icon
- `icon.png` - Default icon
- `icon.icns` - macOS icon
- `icon.ico` - Windows icon
- `icon.svg` - Source SVG

Icon should represent the D&D/fantasy theme (d20 dice, dragon, sword/shield).
Recommended colors: Gold (#c9a227) and dark parchment (#1a1814).
